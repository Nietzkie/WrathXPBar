# WrathXPBar
A lightweight and customizable experience bar addon for World of Warcraft 3.3.5a (Wrath of the Lich King).  WrathXPBar provides a clean, modern-style XP bar inspired by WeakAuras layouts, with detailed leveling statistics such as session time, XP per hour, and estimated time to next level.  Designed to be minimal, informative, and lightweight.
